Контрольна робота по темі “Рядки”

 

3 варіант

 

1. Задано масив слів: type slovo=string[8];

                                      var mas : array [1..6] of slovo;

Надрукувати слова з масиву, які менші від слова ‘pensil’.

2. Вивести слово, яке утвориться із третіх літер всіх слів масиву.

3. Порахувати кількість слів масиву, які містять більші однієї літери ‘а’.

4. Вивести середнє арифметичне значення всіх довжин cлів.
